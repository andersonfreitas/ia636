Function Principal Component Analysis (PCA)

The function iapca computes the principal components transformation matriz of a given data set. The inpu is the data set organized as one row per sample e columns with attributes of each sample. The output are three: the eigenvector matriz, the diagonal of the covariance matriz and the sample mean.

Definição da função

– 
 1 import numpy as np
 2 
 3 def iapca(X):
 4 
 5     n, dim = X.shape
 6     mu = X.mean(axis=0)
 7     X = X - mu
 8 
 9     C = np.dot(X.T,X)/(n-1)  # Covariance matrix
10     e,V = np.linalg.eigh(C) # eigenvalues and eigenvectors of the covariance matrix
11     indexes = np.argsort(e)[::-1] # sorting eigenvalues from largest
12     e  = e [indexes]         # update e and V
13     V = V[:,indexes]
14     return V,e,mu
