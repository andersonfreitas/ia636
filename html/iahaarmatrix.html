Function iahaarmatrix

namespace:ia636
page:iahaarmatrix

Synopse

Kernel matrix for the Haar Transform.

  • A = iahaarmatrix(N)
    • A: Image.
    • N: Double.
– 
 1 from numpy import *
 2 
 3 def iahaarmatrix(N):
 4     from iaerror import iaerror
 5     from iameshgrid import iameshgrid
 6 
 7     n = floor(log(N)/log(2))
 8 
 9     if 2**n != N:
10        iaerror('error: size '+str(N)+' is not multiple of power of 2')
11        return -1
12 
13     z, k = iameshgrid(1.*arange(N)/N, 1.*arange(N))
14     p  = floor(log(maximum(1,k))/log(2))
15     q  = k - (2**p) + 1
16     z1 = (q-1)   / (2**p)
17     z2 = (q-0.5) / (2**p)
18     z3 = q       / (2**p)
19     A  = (1/sqrt(N)) * ((( 2**(p/2.)) * ((z >= z1) & (z < z2))) \
20                               + ((-2**(p/2.)) * ((z >= z2) & (z < z3))))
21     A[0,:] = 1/sqrt(N)
22     return A
